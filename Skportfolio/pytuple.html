Python Tuple

A tuple in Python is an ordered collection of items, similar to a list. However, unlike lists, tuples are immutable, meaning once a tuple is created, its elements cannot be changed. Tuples are defined by enclosing elements in parentheses (), separated by commas.

Creating Tuples:


# An empty tuple
my_tuple = ()

# Tuple of integers
my_tuple = (1, 2, 3)

# Tuple with mixed data types
my_tuple = (1, "Hello", 3.4)

# Tuple without parentheses (tuple packing)
my_tuple = 1, 2, 3

Accessing Tuple Elements:

Accessing elements in a tuple is similar to lists, using indexing and slicing.


my_tuple = ('p', 'y', 't', 'h', 'o', 'n')

# First element
print(my_tuple[0])  # Output: p

# Third element
print(my_tuple[2])  # Output: t

# Last element
print(my_tuple[-1]) # Output: n

# Slice from index 1 to 3
print(my_tuple[1:4]) # Output: ('y', 't', 'h')

Immutability of Tuples:

Since tuples are immutable, you cannot change, add, or remove elements after creation. Attempting to do so will result in a TypeError.


my_tuple = (1, 2, 3)

# This will raise a TypeError
# my_tuple[0] = 5

# You can reassign the tuple to a new one
my_tuple = (4, 5, 6)
print(my_tuple) # Output: (4, 5, 6)
