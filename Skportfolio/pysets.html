Python Sets

A Set is an unordered collection data type that is iterable, mutable, and has no duplicate elements. Python‚Äôs set class represents the mathematical notion of a set. The major advantage of using a set, as opposed to a list, is that it has a highly optimized method for checking whether a specific element is contained in the set.

Creating Sets:


# An empty set
my_set = set()

# Set of integers
my_set = {1, 2, 3}

# Set with mixed data types
my_set = {1.0, "Hello", (1, 2, 3)}

# From a list
my_set = set([1, 2, 3, 2])
print(my_set) # Output: {1, 2, 3} (duplicates removed)

Adding and Removing Elements:

You can add elements using `add()` and `update()`, and remove using `remove()`, `discard()`, `pop()`, or `clear()`.


my_set = {1, 2, 3}

# Add an element
my_set.add(4)
print(my_set) # Output: {1, 2, 3, 4}

# Add multiple elements
my_set.update([5, 6])
print(my_set) # Output: {1, 2, 3, 4, 5, 6}

# Remove an element
my_set.remove(3)
print(my_set) # Output: {1, 2, 4, 5, 6}

# Discard an element (no error if not found)
my_set.discard(7)
print(my_set) # Output: {1, 2, 4, 5, 6}
