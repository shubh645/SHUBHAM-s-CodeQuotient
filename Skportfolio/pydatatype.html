Python datatype

Data type is nothing but the type used to store the values of different variables. Datatypes of variables varies with the required byte space.

Datatypes are as follows:

  • Numeric:
    • int (Integer): 2, -2
    • float (Float): 2.2, 19.1
    • complex (Complex): 2i + 1, 8i
  • String:
    • str (String): "Shubham"
  • Boolean:
    • True
    • False
