Python Dictionary

A dictionary in Python is an unordered collection of data values, used to store data values like a map, which, unlike other Data Types that hold only a single value as an element, holds key:value pair. Key-value pair is provided in the dictionary to make it more optimized. Each key-value pair in a Dictionary is separated by a colon (:), whereas each key is separated by a ‚Äòcomma‚Äô.

Creating Dictionaries:


# An empty dictionary
my_dict = {}

# Dictionary with integer keys
my_dict = {1: 'apple', 2: 'ball'}

# Dictionary with mixed keys
my_dict = {'name': 'John', 1: [2, 4, 3]}

# Using dict() constructor
my_dict = dict({1:'apple', 2:'ball'})

Accessing Dictionary Elements:

Elements are accessed using their keys.


my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

# Accessing value using key
print(my_dict['name']) # Output: John

# Using get() method
print(my_dict.get('age')) # Output: 30
print(my_dict.get('country')) # Output: None

Modifying Dictionaries:

Dictionaries are mutable. You can add new key-value pairs, change values, or remove pairs.


my_dict = {'name': 'John', 'age': 30}

# Add a new key-value pair
my_dict['city'] = 'New York'
print(my_dict) # Output: {'name': 'John', 'age': 30, 'city': 'New York'}

# Change a value
my_dict['age'] = 31
print(my_dict) # Output: {'name': 'John', 'age': 31, 'city': 'New York'}

# Remove a key-value pair
del my_dict['city']
print(my_dict) # Output: {'name': 'John', 'age': 31}
