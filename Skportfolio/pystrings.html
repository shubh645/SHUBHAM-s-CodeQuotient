Python Strings

Strings are sequences of characters. They are immutable, meaning once created, their contents cannot be changed. Strings are enclosed in single quotes '' or double quotes "".

Creating Strings:


# Single-line string
my_string = 'Hello, Python!'

# Multi-line string
multi_line_string = """This is a
multi-line string."""

# String with double quotes
another_string = "Data Analysis"

Accessing Characters:

Characters can be accessed using indexing and slicing, similar to lists and tuples.


my_string = "Python"

# First character
print(my_string[0]) # Output: P

# Last character
print(my_string[-1]) # Output: n

# Slice from index 1 to 3
print(my_string[1:4]) # Output: yth

String Operations:

Strings support various operations like concatenation, repetition, and built-in methods.


str1 = "Hello"
str2 = "World"

# Concatenation
print(str1 + " " + str2) # Output: Hello World

# Repetition
print(str1 * 3) # Output: HelloHelloHello

# Length of string
print(len(my_string)) # Output: 6

# Convert to uppercase
print(my_string.upper()) # Output: PYTHON

# Replace characters
print(my_string.replace('o', '0')) # Output: Pyth0n
