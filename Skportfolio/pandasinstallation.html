Introduction

Pandas Installation

To install Pandas, by using pip if you already installed python in your system then just simply open the terminal and type:

pip install pandas

Verifying Installation

To verify that Pandas has been installed correctly, you can run the following command in your Python environment:

import pandas as pd

Using Pandas

Once Pandas is installed, you can start using it in your Python scripts. Here is a simple example:

import pandas as pd

# Create a DataFrame
data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35]
}
df = pd.DataFrame(data)

# Display the DataFrame
print(df)

This creates a DataFrame with specified data and prints it to the console.

Output:

    Name    Age
0  Alice     25
1    Bob     30
2 Charlie    35
